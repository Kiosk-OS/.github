# Security Policy

## Supported Versions

KioskOS is currently in early development. Security updates are provided for the latest public release and the current main branch.

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues, discussions, or pull requests.

To report a vulnerability, email:

**kioskos@proton.me**

Please include as much detail as possible, including:

- A description of the vulnerability
- Steps to reproduce it
- Affected version, commit, or image build
- Any relevant logs, screenshots, or proof-of-concept details
- Whether the issue is already public or being actively exploited

We will try to acknowledge valid reports within 7 days.

If the vulnerability is accepted, we will work on a fix and may coordinate disclosure timing with the reporter when appropriate. If the report is declined, we will explain why when possible.

Please avoid publicly disclosing the vulnerability until a fix or mitigation is available.
