## KioskOS

KioskOS is a lightweight, free and open-source operating system for building clean, reliable kiosk displays.

Flash it, boot it, connect to a network, and get a locked-down Chromium kiosk environment without fighting the usual desktop clutter.

## Features
- Raspberry Pi kiosk image
- First-boot onboarding
- Wi-Fi, Ethernet, timezone, device, and SSH setup
- Chromium kiosk mode
- Touch-friendly interface
- Hidden cursor support
- Local app/display deployment over SSH
- Built for simple and reliable kiosk setups

## Status & Plans

KioskOS is early and actively being built. Expect changes, rough edges, and rapid iteration.

KioskOS is planned to expand beyond the initial Raspberry Pi image over time. Possible future work includes:
- Armbian-based images for broader single-board computer support
- A built-in on-screen keyboard for easier touchscreen-only setup and to make people's kiosks work easier
- A desktop companion app for easier deployment and management over SSH
- Better local web app installation and update workflows
- More polished documentation, release images, and setup guides
